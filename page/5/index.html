tcpdump常用命令

  • -A 以ascii的格式输出
  • -r 读文件
  • -w 写文件
  • -c 指定多少个包后退出tcpdump
  • -D 输出可用的网络接口
  • -e 可用于输出MAC地址信息
  • -i 指定接口
  • -n 不将ip地址和端口转成名称
  • -t 不输出时间戳
  • -x 输出数据包的二进制码
  • -v verbose 输出全部的协议编码
  • host 指定只抓取某主机数据
  • port 指定端口
  • src 指定源
  • dst 指定目的地
  • 过滤数据 例如：直接将icmp追加到尾部过滤出icmp的数据

《人工智能 一种现代化方法》

人工智能四类定义

像人一样行动 合理地思考
像人一样思考 合理的行动

左侧的定义根据与人类表现的逼真度来衡量成功与否；右侧定义依据合理性的理想表现量来衡量。

像人一样行动， 计算机须具备以下能力：

  • 自然语言处理（natural languageprocessing）使之能成功的用语言交流。
  • 知识表示（knowledge representation）以存储它知道的或听到的信息。
  • 自动推理（automated reasoning）以运用存储的信息来回答问题或得出结论。
  • 机器学习（machine learning）以适应新情况并检测和预测模式。
  • 计算机视觉（computer vision）以感知物体。
  • 机器人学（robotics）以操纵和移动对象。

像人类一样思考： 认知建模的途径

  • 通过内省-试图捕获我们自身的思维过程。
  • 通过心理实验-观察工作中的一个人。
  • 通过脑成像-观察工作中的头脑。

合理地思考：“思维法则”的途径

  • 逻辑定式。不太靠谱。

合理地行动：合理Agent的途径（重点）

Agent就是能够行动的某种东西。
合理Agent是一个为了实现最佳结果，或者，当存在不确定性时，为了实现最佳期望结果而行动的Agent。
合理的Agent的途径与其他途径相比有两个优点。首先，它比“思维的途径”更一般因为正确的推理只是实现合理性的几种可能的机制之一。其次，它比其他基于人类行为或人类思维的途径更经得起科学发展的检验。

  • 思想如何从物理的大脑中产生？

  • 知识如何导致行动？

  • 什么是能导出有效结论的形式化规则？
  • 什么可以被计算？
  • 我们如何用不确定的信息来推理？

经济学

  • 我们应该如何决策以便收益最大？
  • 当其他人不合作时我们应该如何做到这样？
  • 当收益遥遥无期时我们应该如何做到这样？

神经科学

  • 大脑如何处理信息？

心理学

  • 人类和动物如何思考与行动？

计算机工程

  • 我们如何才能建造高效的计算机？

控制论

  • 人工制品可以如何在其自身的控制下运转？

语言学

  • 语言与思维如何关联？

原则 Ray Dalio

生活原则（一） 拥抱现实，应对现实

把生活想象成一场游戏，我面临的每一个问题都是一个需要破解的迷。我通过揭秘获得一块宝石，即一项原则。它能够帮助我在未来避免同样的问题
体会从犯错中学习的愉悦感

做一个超级现实的人

梦想+现实+决心=成功的生活。从现实出发。有效的工作，提高自己的工作效率。极力避免做无用功。

真相（或者更确切的说，对现实的准确理解）是任何良好结果的根本依据

做到头脑极度开放、极度透明

对于快速学习和有效改变而言，头脑极度开放、极度透明是价值无限的。
不要担心别人的看法，使之成为你的障碍。你必须以自己认为最好的独特的方式行事，这样做肯定有反馈，你必须开明的思考这些反馈。
拥抱极度求真和极度透明将带来更有意义的工作和更有意义的人际关系。

观察自然，学习自然规律

宇宙浩瀚无垠，人类不过是沧海一粟。从大自然中学习如何认识现实规律。
不要固守你对事物应该是什么样的看法，这将使你无法了解真实的情况。 大多数人根据事物对自身影响判断好坏。正在运行的自然进程，它能够有效的促进整个世界的改善，比任何程序都要好。
一个东西要“好“，就必须符合现实规律，并促进整体的进化，这能带来最大的回报。
进化是宇宙中最强大的力量，是唯一永恒的东西，是一切的驱动力。
不进化就死亡。不断学习新知识，把握新规律。

进化是生命最大的成就和最大的回报

个体的激励机制必须符合群体的目标。
为了整体趋向最优化，而不是个体。
通过快速试错以适应现实是无价的。
意识到你既是一切又什么都不是，并决定你想要成为的样子。自下而上的看，自己就是一切，如果生命终止了，对我们而言，世界也就消失了。自上而下的看，人类都不过是沧海一粟，个体生命的终结更加虚无。宇宙运行自有其规律。
你的未来取决于你的视角

理解自然提供的现实教训

把你的进化最大化。对大多数人而言，成功就是以尽可能高效的方式拼搏和进化，即迅速认识自身和所处环境，然后通过改变实现进步。
记住“没有痛苦就没有收获”。自然让我们痛苦是有意图的。痛苦让我们清醒，帮助我们进步。
自然的一项根本法则是，为了赢得力量，人必须努力突破极限，而这是痛苦的。

痛苦 + 反思 = 进步

生活不可避免的给你考验， 而你愿不愿意以退为进、从失败中获益，取决于你自己。
迎接而非躲避痛苦。找到、接受，并学会如何应对你的弱点；更喜欢周围的人对你坦诚，而不是隐藏对你的负面看法；展现真实的自我，而不是强行把弱点伪装成优势。
接受严厉的爱。不断寻找、实现和追求更大胆的目标，不仅适用于个人和社会如何前进的问题。它同样适用与逆境，而逆境是不可避免的。你一生中必定会经历某个一败涂地的时刻。你可能工作失败、家庭失败、失去亲人、遭遇严重事故或疾病，或者发现你想要的生活永远遥不可及。总会有某些厄运找上你。那时，你将深陷痛苦痛苦，可能觉得失去了鸡血前进的力量。然而，你几乎总是有这样的力量-你最终是否成功，取决于你能否认识到这个事实，尽管可能但是看起来不是这样。

考虑后续与再后续的结果

选择自己真正想要的东西，顶住诱惑，克服可能妨碍自己实现目标的痛苦，这样的人拥有成功人生的可能性要大得多。

接受结果

不管在生活中遇到什么情况，如果你能负起责任，进行良好的决策，而不是抱怨你无法控制的东西，你将更有可能成功找到幸福。心理学家称此为内控点，各种研究不断显示，拥有内控点的人比其他人做的更好。所以不要为喜不喜欢自己的处境担忧。生活根本不关心你喜欢什么。你必须根据自己的愿望找到实现愿望的途径，然后鼓起勇气坚持下去。

从更高的层次俯视机器

人类拥有独特的从更高层次俯视的能力，这不仅适用于理解现实和现实背后的因果关系，也适用于俯视自身和周围的人。我把这种超越自己和其他人的处境并客观的俯视处境的能力，称为“更高层次的思考”。更高层次的思考能让你学习和影响生活中的各种因果关系，利用这些因果关系得到你想要的结果。
把自己想象成一部在大机器里运转的小机器，并明白你有能力改变你的机器以实现更好的结果。你有你的目标。你用来实现目标的方式，我称为你的机器。它包括设计（必须做的事）和人（由谁来做这些必须做的事）。人包括你和帮助你的人。
通过比较你实现的结果和你的目标，你就能确定如何改进你的机器。
区别作为机器设计者的你和作为机器中工作者中的你。让自己成为自身生活的良好设计者和管理者，而不是机器中工作者。想要取得成功，作为设计者和管理者的你必须客观看待作为工作者的你，不高看自己，也不让自己承担不应该承担的任务。大多数人都是凭情感行动，只看眼前。而没有从上述战略性的视角出发。他们的生活充满了没有方向性的情感体验，随波逐流。如果你想在晚年回顾一生时觉得实现了人生愿望的话，你就不应该这样做。
大多数人犯下的最大错误是不客观看待自己以及其他人，这导致他们一次次的栽在自己或其他人的弱点上。
成功的人能超越自身，客观看待事物，并管理事物以塑造自身。
在你不擅长的领域请教擅长的其他人，这是一个你无论如何都应该培养的出色技能，这将帮助你建立起安全护栏，避免自己做错事。
因为客观看待自身很困难，所以你需要依赖其他人的意见，以及全部证据。
如果你的头脑足够开放，足够有决心，你几乎可以实现任何愿望。如果一条道路走不通，走另一条道路。
总结：

  • 不要混淆你的愿望和事实
  • 不要为自身形象担忧，只需关心能不能实现你的目标。
  • 不要过于重视直接结果而忽视后续、再后续的结果。
  • 不要让痛苦妨碍进步。
  • 不要把不好的结果归咎于任何人，从自己身上找原因。
