tcpdump常用命令

  • -A 以ascii的格式输出
  • -r 读文件
  • -w 写文件
  • -c 指定多少个包后退出tcpdump
  • -D 输出可用的网络接口
  • -e 可用于输出MAC地址信息
  • -i 指定接口
  • -n 不将ip地址和端口转成名称
  • -t 不输出时间戳
  • -x 输出数据包的二进制码
  • -v verbose 输出全部的协议编码
  • host 指定只抓取某主机数据
  • port 指定端口
  • src 指定源
  • dst 指定目的地
  • 过滤数据 例如：直接将icmp追加到尾部过滤出icmp的数据
